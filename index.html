      Hướng dẫn nhanh
      1. Trong SketchUp, Export sang .dae hoặc .gltf rồi import vào đây.
      2. Click các mặt LED trong khung 3D, hoặc bấm Tự động phát hiện LED để hệ thống đoán theo tên/material.
      3. Chuyển sang tab 2D Mapping, tải ảnh Mapled tham chiếu rồi kéo các tấm LED khớp với mapled.
      4. Chỉnh kích thước thật, pixel pitch, độ phân giải; sau đó Export file JSON cấu hình.
